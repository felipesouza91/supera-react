## Desafio Supera UI

Desafio do processo seletivo FullStack Java-React

## Tecnologias utilizadas

- React
- React Hook Forms
- Axios
- Tailwind CSS

## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  


To Start Server:

`npm run dev`  

To Visit App:

`localhost:3000`
